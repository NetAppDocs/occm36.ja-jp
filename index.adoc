= Cloud Manager および Cloud Volumes ONTAP のドキュメント
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs:

OnCommand Cloud Manager では、 NetApp Cloud Volumes ONTAP を導入して管理できます。 NetApp 解決策は、クラウドベースのワークロードに対して保護、可視化、制御を提供するデータ管理 です。
